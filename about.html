FocusForge

A minimal desktop app for planning, running, and analyzing your deep work sessions.

Tauri React TypeScript Tailwind CSS Recharts macOS / Windows
Overview

What is FocusForge?

FocusForge is a Pomodoro-style productivity desktop app built for developers, students, and knowledge workers who want more than just a timer — they want to understand where their time goes.

⏱

Pomodoro Timer

Focus, short break, and long break modes with full start, pause, reset, and skip controls.

🎛

Custom Durations

Set your own focus and break lengths — 25/5, 50/10, 90/15, or anything in between.

🏷

Session Tags

Tag each session with a project or activity and filter your history to see exactly how your time is spent.

📊

Analytics Dashboard

Today's focus time, completion rate, weekly trend chart, and time broken down by tag — all in one view.

🔔

Native Notifications

Desktop notifications fire when a session ends, keeping you on track without needing to watch the timer.

🌗

Light / Dark / System

Three theme modes with automatic detection so the app always matches your environment.

Why Desktop?

A native app, not a tab

A browser tab competes with every other tab. A native desktop app runs on your own terms.

1
System-level notifications

Session transitions fire as real OS notifications — no browser permission dialogs, no silent failures.

2
Fully offline

All data lives locally. No account, no cloud, no subscription required.

3
Premium feel

Tauri gives the app a native window, native menus, and a binary that launches instantly — not a browser wrapped in Electron bloat.

4
Future-proofed for blocking

The desktop platform makes it possible to add app and website blocking in V2 — something a web app simply cannot do.

Tech Stack

Built with the right tools

Tauri keeps the binary small and fast. React and TypeScript handle the UI. Tailwind and Recharts handle style and data visualization.

Tauri — native desktop shell
React — component UI
TypeScript — type safety
Tailwind CSS — utility styling
Recharts — analytics charts
localStorage — local persistence
